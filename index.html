Things to know before clicking start button!

  • Move the pacman using the arrow keys or the on-screen arrow buttons.
  • The game will begin after the intro music has been played.
  • When colliding with the enemy,the hit animation will be played, After which the player will have 2 seconds of invulnerability.
  • Enter the same name if you want to update your previous score or click cancel to remain anonymous
  • For the best and intended experience, please use Google Chrome with default zoom (100%).
  • If any unexpected error occurs, please refresh your browser and try again.
